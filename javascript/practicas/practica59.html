Suponga que i = 1, j = 2, k = 3 y m = 2. ¿Qué es lo que imprime cada una de las siguientes instrucciones? ¿Son necesarios los paréntesis en cada caso?

a) document.writeln( i == 1 );
R: true

b) document.writeln( j == 3 );
R: false

c) document.writeln( i >= 1 && j < 4 );
R: true

d) document.writeln( m <= 99 && k < m );
R: false

e) document.writeln( j >= i || k == m );
R: true

f) document.writeln( k + m < j || 3 - j >= k );
R: false

g) document.writeln( ! ( k > m ) );
R: false
