Identifique y corrija los errores en cada una de las siguientes instrucciones

a) if (c < 7);
window.alert("c es menor que 7");
R: El if no lleva punto y coma al final, porque seria una linea vacia que no ejecuta nada, lo correcto seria eliminar el punto y coma

b) if (c => 7)
window.alert(c es igual o mayor que 7);
R: El operador esta mal escrito, lo correcto seria >=
