Indique si cada uno de los siguientes enunciados es verdadero o falso, si es falso, explique por qué

a) Para hacer referencia a una ubicación o elemento específico en un arreglo, especificamos el nombre del arreglo y el valor del elemento
R: Verdadero

b) La declaración de una variable reserva espacio para un arreglo
R: Verdadero

c) Para indicar que deben reservarse 100 ubicaciones pra el arreglo entero p, el programador debe escribir la siguiente declaración:
p[100];
R: Falso, debe ser:
var p = new Array(100);

d) Un programa de JavaScript que inicializa los elementos de un arreglo de 15 elementos en cero debe contener cuando menos una instrucción for
R: Verdadero

e) Un programa de JavaScript que calcula el total de los elementos de un arreglo bidimensional debe contener instrucciones for anidadas
R: Verdadero
