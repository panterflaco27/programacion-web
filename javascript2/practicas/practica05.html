Encuentre el o los errores en cada uno de los siguientes segmentos de programa y corrijalos

a)
var b = new Array(10);
var longitud = b.length
for (var i = 0; i <= longitud; ++i)
{
b[i] = 1;
}
R: se hace referencia a un elemento que esta fuera de los límites del arreglo, la solución seria cambiar el <= por un <

b)
var a = [[1, 2], [3, 4]];
a[1, 1] = 5;
