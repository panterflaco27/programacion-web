¿Es posible distinguir un color a partir de su código RGB hexadecimal? Vea el código hex y adivine su color. Para ver a qué color corresponde, mueva el ratón sobre el código hexadecimal. Si quita el ratón de la celda de la tabla de código hex se mostrará el nombre del color.

  • #0000FF
