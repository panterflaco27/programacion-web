Java How to Program, 9/e

De Java How ti Program, 9/e:

Características:

  • Extensa cobertura de los fundamentos: dos capítulos sobre intruccines de control.
  • Enfoque en ejemplos del mundo real.
  • Conjunto de ejercicios Hacer la diferencia
  • Introducción desde los primeros capítulos a las clases, objetos, métodos y cadenas.
  • Manejo de excepciones integrado.
  • Archivos, flujos y serialización de objetos.
  • Secciones modulares opcionales sobre lenguaje y cacterísticas de la biblioteca de la nueva versión Java SE 7.
  • Otros temas: Recursividad, búsqueda, ordenamientos, colecciones genéricas, genéricos, estructuras de datos, applets, multimedia, multihilos, bases de datos/JBDC™, desarrollo de aplicaciones Web, servicios Web y un caso de estudio opcional de Diseño orientado a objetos con un ATM.
