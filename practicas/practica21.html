Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Los navegadores que despliegan el tipo input color como campo de texto requieren que el usuario introduzca el nombre de un color
R: Falso, utiliza código hexandecimal, RGB y HSL

b) Cuando un ususario introduce datos en un formulario y luego lo envía (por lo general, al hacer clic en el botón Enviar), el navegador verifica de inmediato que los datos sean corectos
R: Verdadero

c) HTML5 puede validar si la dirección de correo electrónico introducida por el usuario existe en realidad
R: Falso, puede escribirse cualquier dirección siempre y cuando cumpla el formato

d) Puede agregar required a cualquiera de los tipos input
R: Verdadero

e) Puede habilitar autocomplete sólo para ciertos elementos input
R: Verdadero

f) El elemento time nos permite identificar una fecha, hora o ambas
R: Verdadero

g) El elemento caption proporciona una leyenda para la imagen en un elemento figure
R: Falso, para poner leyendas en figure se utiliza figcaption

h) El elemento details muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento summary
R: Falso, es al revés, summary muestra una flecha con una leyenda y al dar clic se despliega el contenido de details

i) El elemento footer describe el contenido que por lo general aparece en la parte inferior del contenido o del elemento section
R: Verdadero

j) El elemento highlight nos permite resaltar texto
R: Falso, para resaltar un texto se usa mark
