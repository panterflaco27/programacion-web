Complete las siguiente oraciones:

a) El tipo input color permite al usuario introducir un color. Al momento de escribir este libro, la mayoría de los navegadores despliegan el tipo input color como un campo de texto en donde el usuario puede introducir un__código hexadecimal__

b) El atributo__placeholder__permite colocar texto temporal en un campo de texto

c) Si desea anular la validación, puede agregar el atributo formnovalidate al tipo input__submit__

d) El atributo__required__obliga al usuario a introducir valores antes de enviar el formulario

e) Por lo general se muestra el control__spinner__para el tipo input number e incluye sólo los números validos

f) El tipo input__time__permite al usuario introducir una hora, minuto, segundo y fración de segundo

g) El elemento__datalist__porporciona opciones de entrada para un elemento input text

h) El elemento__article__describe contenido que está separado del contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artículo de noticias, mensaje de foro o entrada de blog

i) El elemento__footer__describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento section
