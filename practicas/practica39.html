Indique si cada uno de los siguientes enunciados es verdadero o falso. si es falso, explique porqué

a) La regla @font-face especifica que se utilizará una fuente incrustada cuando el documento se despliegue en una pantalla de computadora
R: Falso, para utilizar una fuente incrustada se usa @media screen

b) Podemos usar gradientes en cualquier propiedad que acepte imagen
R: Verdadero

c) Un gradiente horizontal cambia en forma gradual de arriba hacia abajo
R: Falso, los gradientes horizontales cambian de izquierda a derecha

d) Podemos agregar líneas entre columnas mediante el uso de la propiedad column-gap
R: Falso, para agregar líneas se usa column-rule

e) La regla @media determina el tipo y tamaño del dispositivo en el que se despliega la página. Cuando el navegador examina la regla, el resultado es verdadero o falso. Los estilos de la regla se aplican sólo si el resultado es falso
R: Falso, @media se aplica sólo si el resultado es verdadero

f) Para agregar múltiples imágenes de fondo en un elemento, usamos la propiedad background-position para especificar en dónde colocar cada imagen mediante el uso del modelo de cajas
R: Falso, para colocar cada imagen se usa background-origin
