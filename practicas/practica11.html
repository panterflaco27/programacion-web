Indique cuál de los siguientes enunciados es verdadero y cuál es falso.

a) Un documento de HTML5 válido no puede contener letras mayúsculas en nombres de elementos
R: falso, si se pueden escribir mayúsculas en los nombres

b) Los documentos de HTML5 pueden tener la extensión .htm
R: verdadero

c) &less; es la referencia a entidad de carácter para el carácter menor que (<)
R: falso, para poner < se utiliza &lt;

d) En un documento de HTML5 válido, <li> puede anidarse dentro de etiquetas <ol> o <ul>
R: verdadero
